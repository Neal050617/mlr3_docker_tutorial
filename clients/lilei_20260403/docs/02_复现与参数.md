```bash

bash lina_20251208/20260403_lilei/scripts/run_formal_analysis_corrected.sh all \
  20260403_lilei/reproduce_formal_case_positive_manual
```

该 wrapper 会进入 `lina_20251208/` 后运行，因此输出路径使用 `20260403_lilei/...`。

## SHAP 复现

正式 SHAP 输出目录名：

```text
step3_shap_formal_case_control_force_20260527
```

批量 SHAP 可从仓库根目录运行：

```bash
root="lina_20251208/20260403_lilei/reproduce_formal_case_positive_fullrerun_20260527_0156"
SHAP_OUTDIR_NAME=step3_shap_formal_case_control_force_20260527 \
SHAP_DATASETS=train,test,valid \
SHAP_WATERFALL_GROUPS=case,control \
SHAP_WATERFALL_PER_GROUP=1 \
SHAP_LOCAL_DISPLAY_FEATURES=10 \
SHAP_FORCE_PLOTS=TRUE \
MLR3_ROOT=scripts \
bash scripts/run_mlr3_shap_all.sh "$root"
```

`SHAP_LOCAL_DISPLAY_FEATURES=10` 只限制 waterfall/force 单样本图的展示特征数。当前正式 PDF 中，importance、beeswarm、waterfall 和 force 只对较长特征名自动换行，默认超过 28 个字符时优先在靠近中点的 `-` 后断为最多两行；短特征名不换行；force 以较低的箭头形状单行正负累积贡献条展示 Top 10 特征，SHAP 值标在箭头内，Positive 特征名按箭头起点对齐，Negative 特征名按箭头尾部右对齐。SHAP Excel 仍保留完整数值。

指定一个或多个样本：

```bash
SHAP_WATERFALL_SAMPLES=train:SAMPLE_A,test:SAMPLE_B \
SHAP_FORCE_PLOTS=TRUE \
bash scripts/run_mlr3_shap_all.sh "$root"
```

样本名前缀支持 `train:`、`test:`、`valid:`。valid 数据不存在时会跳过 valid 样本。
